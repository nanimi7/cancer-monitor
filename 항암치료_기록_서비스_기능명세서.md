# 항암치료 기록 웹 서비스 기능명세서

## 서비스 개요
- **형태**: 웹 서비스
- **대상 사용자**: 항암치료 과정을 기록하고 요약하고 싶은 사람
- **목적**: 항암치료 과정의 체계적 기록 및 AI 기반 증상 요약 제공

---

## 주요 기능

### 1번 메뉴: 사용자 정보 등록

#### 기능 설명
사용자의 기본 정보를 등록하고 관리하는 기능

#### 입력 항목

| 항목 | 입력 방식 | 제한 사항 | 필수 여부 |
|------|----------|----------|----------|
| 닉네임 | 텍스트 직접 입력 | 공백 포함 10자 이내, 중복 검사 안 함 | 필수 |
| 생년월일 | Date Picker | 년/월/일 선택 | 필수 |
| 성별 | 라디오 버튼 | 남/여 중 1개 선택 | 필수 |
| 병명 | 텍스트 직접 입력 | - | 필수 |
| 기저질환 정보 | 텍스트 직접 입력 | 공백 포함 1000자 이내 | 선택 |

---

### 2번 메뉴: 처방받은 약 목록 관리

#### 기능 설명
사용자가 복용 중인 약물 정보를 등록, 조회, 삭제할 수 있는 기능

#### 하위 기능

##### 1) 약물 등록하기
- **입력 항목**
  - 약물 이름 (필수): 텍스트 입력
  - 복용 횟수 (필수): 텍스트 입력
  - 주요 효능 (필수): 텍스트 입력
  - 기타 내용 (선택): 텍스트 입력

- **동작 방식**
  - (등록하기) 버튼 클릭 시 약물 정보 저장
  - 등록 완료 후 목록 화면으로 자동 이동

##### 2) 약물 목록 조회
- **표시 방식**: 표 형태로 리스트업
- **표시 정보**: 약물 이름, 복용 횟수, 주요 효능, 기타 내용
- **추가 기능**: 각 약물 정보 우측에 (-) 삭제 버튼 제공

##### 3) 약물 삭제하기
- **동작 방식**
  1. (-) 버튼 클릭
  2. "삭제하시겠습니까?" 팝업 노출
  3. (확인) 클릭 시 영구 삭제

### 3번 메뉴: 일일 증상 기록 캘린더

#### 기능 설명
매일매일의 증상을 캘린더 형식으로 기록하고 조회할 수 있는 기능

#### 목록 화면 (월간 캘린더)

##### 표시 방식
- 월간 캘린더 형식으로 제공
- 내용이 있는 날짜: 레드닷 아이콘 표시 또는 날짜 칸 색칠로 입력 유무 구분
- 입력된 값 중 다음 정보를 뱃지 형식으로 표시:
  - 항암 진행 횟수
  - 항암 회차
  - 주요 부작용

#### 날짜 클릭 시 동작

##### 신규 작성 시
- "입력된 내용이 없습니다." 텍스트 표시
- (작성하기) 버튼 제공

##### 기존 내용 있는 경우
- 작성된 내용을 템플릿 형식으로 정리하여 노출
- (수정하기) 버튼 제공

#### 작성/수정 화면 템플릿

| 항목 | 입력 방식 | 제한 사항 | 필수 여부 | Placeholder |
|------|----------|----------|----------|------------|
| 날짜 | Date Picker | 년/월/일 선택 | 필수 | - |
| 항암 진행 횟수 | 텍스트 입력 | 공백 포함 10자 이내 | 필수 | "몇 번째 항암인지 써주세요 ex.1차/2차.." |
| 항암 회차 | 텍스트 입력 | 공백 포함 10자 이내 | 필수 | "동일 차수 내에서 몇 번째인지 써주세요 ex.1회차/2회차.." |
| 항암 진행 일차 | 텍스트 입력 | 공백 포함 10자 이내 | 필수 | "동일 회차 내에서 며칠 차인지 써주세요 ex.3일차" |
| 식사량 | 텍스트 입력 | 공백 포함 1000자 이내 | 필수 | - |
| 음수량 | 텍스트 입력 | 공백 포함 1000자 이내 | 필수 | - |
| 운동량 | 텍스트 입력 | 공백 포함 1000자 이내 | 필수 | - |
| 주요 부작용 | 체크박스 (복수 선택 가능) | 구토, 오심, 발열, 손발저림, 두통, 설사, 변비 | 필수 | - |
| 주요 증상 | 텍스트 입력 | 공백 포함 5000자 이내 | 필수 | - |

- **저장 버튼**: 하단에 (등록하기) 버튼 제공
- **동작**: 클릭 시 입력된 정보 저장 및 확인 가능

### 4번 메뉴: AI 요약

#### 기능 설명
Claude 또는 GPT 등 AI를 활용하여 항암 회차별 증상을 요약하여 제공

#### 데이터 그룹핑 기준
- 항암 진행 횟수
- 항암 회차 정보
- 항암 진행 일차

#### 사용자 인터랙션
1. 사용자가 항암 진행 횟수 선택
2. 사용자가 항암 회차 정보 선택
3. 선택된 기간 기준으로 AI 요약 제공

#### 요약 내용
선택된 기간의 다음 항목들을 분석하여 요약:
- 식사량
- 음수량
- 운동량
- 주요 부작용
- 주요 증상

#### 요약 예시
**선택: 1차, 1회차**

해당 기간에 입력된 전체 항암 진행 일차 정보를 그룹핑한 후, 다음과 같은 정보를 요약 제공:
- 식사량 패턴 및 변화
- 음수량 패턴 및 변화
- 운동량 패턴 및 변화
- 주요 부작용 발생 빈도 및 특이사항
- 주요 증상 특이사항
- 의료진에게 전달할 만한 주요 정보

**출력 예시**:
"1차, 1회차 동안의 식사량은 평균 50% 수준이었으며, 주요 부작용은 구토가 3회, 오심이 5회 발생했습니다. 다음 진료 시 의사에게 전달하면 좋은 내용: 3일차부터 식사량이 급격히 감소했으며, 손발저림 증상이 지속적으로 나타났습니다."
